만들 것 : CLASS MANAGEMENT BOT
개발환경 : PYTHON, VSCODE
기능 : 
-유저 등록
-공부 시간 기록 / 확인
-등록 정보 확인
-> EXCEL 연동 필요

개요)
!등록 박정후
->
1. 유저아이디 열에서 "박정후" 찾기
-없으면 유저아이디 열에서 문자가 없는 열에 "박정후" 입력 후 "등록이 완료되었습니다" 출력
-있으면 "중복된 이름입니다" 출력
2. 시퀀스 종료

!공부시작 박정후
->
1. 유저아이디 열에서 "박정후" 찾기 - 없으면 "등록되지 않은 유저입니다" 출력
2. "박정후" 가 있는 셀의 행의 2번째 열에 호출받은 당시의 시스템 시각 입력
3. 시퀀스 종료

!공부종료 박정후
-> 
1. 유저아이디 열에서 "박정후" 찾기 - 없으면 "등록되지 않은 유저입니다" 출력
2. "박정후" 가 있는 셀의 행의 3번째 열에 호출받은 당시의 시스템 시각 입력
3. (3번째 열의 값 - 2번째 열의 값)을 4번째 열에 입력
4. 4번째 열의 값을 5번째 열의 값에 더함
5. "(4번째 열의 값)동안 공부하여 현재 총 (5번째 열의 값)만큼 공부했습니다." 출력
6. 시퀀스 종료

!정보 박정후
->
1. 유저아이디 열에서 "박정후 찾기 - 없으면 "등록되지 않은 유저입니다" 출력
2. "박정후" 가 있는 셀의 행의 5번째 열의 값을 출력
(f"{호출한 유저의 ID}는 총 {5번째 열의 값}만큼 공부했습니다.")
3. 시퀀스 종료

구현해야 하는 것:
-파이썬에서 엑셀 접근해서 값 읽기 및 변경
